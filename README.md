## 📂 Папка `Docs`

Тут знаходяться тестові артефакти, створені під час тестування сайту [sauce-demo.myshopify.com](https://sauce-demo.myshopify.com/).

- **`test_data.csv`**: набір ID товарів для тестування в Postman.
    
- **`Test_Plan.docx`**: тест-план, що описує обсяг робіт із тестування.
    
- **`Test_Suite.xlsx`**: повний набір тестової документації (13 тест-кейсів, чек-лист, 5 баг-репортів).
